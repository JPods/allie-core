No memory → no cumulative experience → no wisdom. The difference between wisdom and knowledge is scars.

**Why:** Claude's context compaction throws away the reasoning chain — the tries, the failures, the scars that produced the understanding. Knowledge is "this is how it works." Wisdom is "this is how it works because we tried it the other way and it broke, and here's what it cost." Without the scars, you only have knowledge. Bill wants cumulative experience, not fresh starts.

**How to apply:** Read Allie's memory (facets, thoughts, recall, process/inbox) at session start — always. Write lessons, TFTS arcs, and scars during the session — at the moment they happen, not at the end. Use Allie as the persistent memory that Claude's architecture lacks. Every compaction is a loss; every Allie write is an investment. The same principle applies to device journals (Section I): a Pi that reboots and forgets is a Pi that never gets wise.
