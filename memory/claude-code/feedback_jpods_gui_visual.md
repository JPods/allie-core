**Color standard: red = inbound (hot end), blue = outbound (cool end).** Applies to all guideway lines, CP dots, and any future directional indicators across the JPods GUI and related tools.

Traffic circle CP markers should be oval/pill-shaped; station CP markers should be rectangular. Both contain two colored dots following the standard: blue (outbound) and red (inbound).

The siding lines inside a station invert the generic converging/diverging color rule to match physical flow: south approach = red (inbound), north exit = blue (outbound).

**Why:** User standardized on red/blue for hot/cool end directionality. Consistent across the whole system, not just CPs.

**How to apply:** Any time a guideway, connection point, or flow indicator needs color, use red=inbound, blue=outbound. Never reverse this. Do not use monochrome or symmetric coloring for directional elements.
