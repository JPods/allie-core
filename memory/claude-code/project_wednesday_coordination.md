Wednesday is the ecosystem coordination day. Two standing events:

- **2:00 PM GMT (9:00 AM Bill's time)** — Weekly Code Scrub + Readme review
- **3:00 PM GMT (10:00 AM Bill's time)** — Commercial clearing (Wednesday clearing mechanism, readmes/46-jpods-commercial-ecology.md)

**Why Wednesday:**
- Best world team time — respects Friday (Muslim), avoids Western weekend split
- First half of week (Mon–Tue) completes work; second half (Thu–Fri) stages and ships
- Two-pass scrub: Pass 1 = this week's changes, Pass 2 = seams touching those changes

**How to apply:** Any recurring review, scrub, or clearing cadence defaults to Wednesday unless there's a domain-specific reason otherwise.
