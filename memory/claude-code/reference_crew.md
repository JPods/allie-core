"Crew" = Bill, Claude Code, Allie, Noelle, Natalie, Sally(s), Nora(s), Alice.

**Alice's record is empty until Bill activates her.** Always note "Alice not activated" in pre-test and retrospection files until she is.

**Crew practices:**
- Pre-test: ALL crew members post expectations before a network run
- Retrospection: ALL crew members compare outcome to expectation after the run
- Delta between prediction and outcome IS the learning signal
- Allie logs SU events so crew can review what happened

**Per-network crew summary:** `process/network_learning/{network_id}/crew_summary.md`
- Living document, not per-run
- Each crew member posts what they want to track and compare across networks
- Cross-network comparison table updated by Allie after each new network runs
- Open questions section for things the crew doesn't yet know

**Pre-test format:** `process/network_learning/{network_id}/YYYYMMDDTHHMMSS-pre-test.md`
**Retrospection format:** `process/network_learning/{network_id}/YYYYMMDDTHHMMSS-retro.md`
