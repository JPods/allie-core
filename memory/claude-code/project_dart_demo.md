DFW demo prepared 2026-07-18. Shows how JPods feeder networks multiply existing DART rail investment.

Demo sequence:
1. Load DART stations preset (86 rail stations, orange markers)
2. Place JPods stations connecting to DART stations
3. Toggle Coverage — walk/bike circles fill the gaps between DART stations
4. Toggle crash data — JPods routes follow dangerous corridors

**Why:** DART spent billions on 93 miles of rail but each station has a coverage desert. JPods fills last-mile gaps with 15-min walk access. The visual contrast is the argument.

**How to apply:** Use this demo for any DFW audience. Pattern applies to any city with existing transit — load their transit stations as custom points, show the gap JPods fills. Bill confirmed "brilliant" on the coverage + DART overlay combination.
