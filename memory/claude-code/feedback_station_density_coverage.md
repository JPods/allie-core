More small stations beats fewer big ones. Station density is about coverage circles, not capacity.

**Why:** Prime Law of Networks — V ∝ n²/p. Value scales with number of nodes, not size. 50 slots handles 10,000 people/hour. A 15-minute walk to a station is acceptable. Packet size toward 1.

**How to apply:** When placing stations, optimize for coverage (walk circles fill gaps) not throughput. Crash corridors tell Noelle where guideways run. Walk circles tell her where stations are sufficient. Two different signals, one network. Never argue for fewer, bigger stations — always more, smaller ones.
