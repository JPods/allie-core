Create a training video library across all three public-facing programs. Dedicated session(s) required.

**Programs:**
- **JPods3D** (SketchUp plugin) — 8-step workflow, each step is a video; plus station templates, waypoints, Build pipeline, animation controls
- **MeshMobility** — network design, simulation, overlays (AADT, crashes, census, custom points), library browse/clone, keyboard shortcuts
- **WC3** (WebClerk) — DataBrowser, contacts, orders, inventory, sync, Alice features, pricing

**Process per feature:**
1. Review the code to understand what the feature actually does
2. Write a 1-4 minute script (narration + screen actions)
3. Record (Bill or screen capture with voiceover)
4. Host on Vimeo, link from README and docs

**Why:** Users need to see features working, not read about them. Short videos convert visitors to users. Scripts written from code knowledge are accurate; marketing-written scripts miss details.

**How to apply:** Start a dedicated session. Read the relevant program code, list all user-facing features, write scripts in order of the user workflow. Don't mix with development work.
