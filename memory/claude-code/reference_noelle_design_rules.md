Key design rules established 2026-07-05:
- Circles at corridor intersections (4 CPs = junctions), stations mid-block (2 CPs = stops)
- ~1:1 ratio circles to stations; too few circles = disconnected islands
- Primary network: AADT-driven (10,000+ trips/day corridors)
- Secondary network: accident-driven (fatal crash clusters on moderate-AADT 5-10k roads)
- Standard grid: ~0.5 mile (800m) spacing (confirmed across 25+ networks Bill built)
- Stations at commercial nodes on neighborhood BOUNDARIES, not inside neighborhoods
- JPods is Middle-Mile; Last-Mile = walking/biking/scooters/LUVs
- Mesh over capacity: parallel paths so there is NEVER a stoppage
- JPods sells certainty, not speed
- Capacity bottleneck: station dwell (30s/pod), not guideway (9,191 people/hr/direction)
- 50+50 slots per terminus = 18,840 people/hr
