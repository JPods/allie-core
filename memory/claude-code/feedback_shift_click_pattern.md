name: Shift-click as power user pattern
description: Shift-click for destructive or alternate actions — delete layouts, open any model in DataBrowser; Bill loves this pattern
type: feedback
---

Shift-click is the standard modifier for power-user alternate actions in WebClerk UI:
- Shift-click a saved layout → delete it (with confirmation)
- Shift-click any sidebar model → open in DataBrowser instead of dedicated page

**Why:** Zero visual clutter, discoverable by doing, consistent across the app. Bill specifically called it out as "such a cool tool."

**How to apply:** Use shift-click for any future alternate action that would otherwise need a separate button or context menu. Don't add visible delete buttons or mode switches when shift-click can serve the same purpose.
