              on the panels.
    revealed: The solar geometry in the model was built before the instance-tagging fix was applied.
              All existing Solar_* groups and their child instances were still on Untagged (Layer0).
              Hiding "JPods Solar" hid nothing because nothing was assigned to it.

  - try:      Add retag_existing_solar(model) to ensure_tags. Walk top-level model entities for
              Solar_* groups; assign group and child instances to solar_tag if still on Untagged.
              ensure_tags runs on every model open (AppObserver + startup timer) — idempotent.

principle: Two categories of fix are required when a tag assignment is added to a build pipeline:
           (1) the pipeline fix — new geometry gets the right tag going forward;
           (2) the retag pass — existing geometry built before the fix must be reassigned on open.
           Without (2), the fix is correct but invisible until the model is rebuilt from scratch.
           Always ask: "what geometry predates this fix?" and handle it explicitly.
