# sum-claude-recall.md — Aggregated Claude Recall
**Purpose:** Claude's cross-session working knowledge that has been tested and confirmed.
Daily claude-recall.md files are raw — open predictions, pattern hypotheses, pre-loaded
context that may be wrong. This file contains only what held up.

**Curation rule:** An entry earns a place here when a WI prediction was accurate (concern
matched what DNW revealed) OR a pre-loaded pattern correctly anticipated a session's finding,
confirmed across at least two instances. Entries are removed when a session contradicts them.

**Format per entry:**
```
## [Short name] — confirmed [date]
Pattern: one sentence
Evidence: [wi/dnw/tfts files that confirm it]
Applies to: [SU | RT | PH | SYS | WC3 | cross-domain]
Watch for: the edge case most likely to break this pattern
```
