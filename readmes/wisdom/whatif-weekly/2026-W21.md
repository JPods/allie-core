## Why These Items

**C-W21-1 (teleport jumps):** The station-internal gap is the largest known defect in the
map.json animation architecture. High confidence because gap logging already confirmed this
in build_pods output.

**C-W21-2 (length_mm for features):** Feature lines are generated from platform stub geometry
— a different code path than inter-station bezier segments. The mm measurement has not been
explicitly verified on this path.

**C-W21-3 (indirect references to archived tasks):** Archiving by removal is clean, but
console tasks sometimes call shared module methods that were only ever exercised through
the archived task. Low confidence because the kept tasks are tightly scoped and unlikely
to have this dependency.

**C-W21-4 (RT vs map.json length divergence):** Two different systems measuring the same
network for different purposes. Worth documenting as expected behavior rather than a bug.

**C-W21-5 (no restore requests):** The 27 archived tasks were all diagnostic or exploratory.
The core workflow (Build → Validate → Animate) is complete in the 16 kept tasks.
