## Why These Items

**C-W23-1 (arc undersampling resolves cleanly):** The exterior-arc fix (`_generate_uturn_arc_pts_mm` with centroid-based CW/CCW selection) was written and committed in the prior arc of work. The only missing step is running it against the open template. High confidence because the code path is tested; this is a data-population step, not a code fix.

**C-W23-2 (S007/S008 traces to direction preservation):** The proof output shows SEVERE on multiple straight tracks in both models. The same symptom appeared in station_line_end before the direction-preservation problem was identified: pts_mm written in mesh order, endpoint comparison giving wrong results, proof comparing reversed geometry. The root cause is likely identical.

**C-W23-3 (trip panel reused for vehicles):** `showTripSequence` was explicitly designed with a `label` parameter and `_TRIP_PANEL_TASK_IDS` extensibility for this reason. Bill confirmed "We will need this `<div trip>` for vehicles also" in today's session. The pattern is already committed; the only remaining work is wiring the vehicle trip data into it.

**C-W23-4 (Hold Loop trip panel nora_id gap):** `preview_hold_loop_sequence` in Sally takes a station_id and generates a template-level sequence (no vehicle state). When Hold Loop task runs for a specific pod, it has the pod's current slot position — which determines which partial loop it takes. If the Hold Loop run lambda calls the same formatter without adapting for pod state, the preview will show a generic sequence rather than what the specific pod will actually do.

**C-W23-5 (logistics vs. transit framing):** Every JPods pitch so far has been framed around transit — moving people, reducing car trips, last-mile access. The cargo/waste continuous-flow argument is in CLAUDE.md but has not been the lead. Logistics and supply-chain audiences speak in flow rates, sorting yield, dead-head cost, and first-mile/last-mile economics. That's JPods' strongest structural argument and it has not been tested with its natural audience.
