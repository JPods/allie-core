## Lessons for Allie

1. **Hidden faces z-fight** — SketchUp hidden faces still participate in rendering and
   z-fight with coincident faces in adjacent geometry. At gate junctions, two cap faces
   at the same position = visual artifact regardless of hidden state. Erase caps.
   Domain: [SketchUp]

2. **Bounding box extents ≠ gate outer face** — cp component bounding box includes ALL
   sub-geometry including decorative visual indicators. Outward_ext varies 111-296mm
   across station types. Not reliable for gate position detection. If explicit origin
   datum exists (and cp instances were placed with origin at gate), use origin.
   Domain: [SketchUp]

3. **Measurement invalidated by template edit** — The "1.5m off" measurement that
   drove the bounds.center change was made on a template version that predates cp
   instance placement. Geometric measurements on template geometry expire when the
   template is edited. Always note which commit/version a measurement was made on.
   Domain: [SketchUp]

4. **Hardcoded geometry constants need a home** — stub_pair_length_mm = 2000 was
   stale after user changed stubs to 5m. Constants derived from template geometry
   must either be read from template geometry at runtime or flagged when templates change.
   Domain: [SketchUp]
