## Lessons for Allie

1. **map.json is the gap detector** — When `resolve_gw_segs_from_map` logs a map gap, it means
   the Build pipeline has not generated a gw entity for that segment. The log is the action item.
   Do not synthesize geometry to work around it. Domain tag: [SketchUp]

2. **Console task count is a complexity budget** — 43 tasks was noise. 16 is signal. Archive
   aggressively. The archive preserves the code; the console communicates the workflow.
   Domain tag: [Cross-domain]

3. **Permanent geometry > GL overlay for reference paths** — GL Tools are for interactive
   cursor-feedback. Static reference overlays (FollowMe paths, trip paths, routes) belong
   as permanent geometry groups: they persist across tool switches and can be cleared by name.
   Domain tag: [SketchUp]

4. **Three display methods that look the same should use the same helpers** — Shared
   `draw_map_polyline` + `draw_gap_flag` enforce the color standard and gap representation
   consistently across all three display paths. Domain tag: [SketchUp]
