## Lessons for Allie

1. **FIPS codes are inconsistent across APIs** — FCC returns 5-digit county FIPS (state+county), Census and TIGERweb expect 3-digit county-only. Always strip the state prefix. Domain tag: [MeshMobility] [Cross-domain]

2. **NHTSA FARS API is blocked by Akamai WAF** — bulk CSV download works, API doesn't. The ZIPs are ~50MB national files; filter to state after download. Newer years (2021+) have BOM characters in CSV headers — use `utf-8-sig` encoding. Domain tag: [MeshMobility]

3. **FHWA HPMS field names are lowercase** — `aadt` not `AADT_VN`. Service URL pattern: `HPMS_FULL_{ST}_{YEAR}`. Geometry is polylines, not points — extract midpoints. Domain tag: [MeshMobility]

4. **Conservative payback builds credibility** — The math shows 2-3 year payback but behavioral transition is slower. Bill uses 7 years for capital presentations. Car ownership, leases, habits take years to unwind. Aggressive numbers invite skepticism; conservative numbers that are still extraordinary (7-year payback on transit infrastructure is unheard of) build trust. Domain tag: [Cross-domain]

5. **Cloudflare tunnel bypasses the cert.pem problem** — Zero Trust dashboard → Connectors → Create Tunnel gives a token. `sudo cloudflared service install <token>` — no cert.pem needed, no browser callback, auto-starts on boot. Domain tag: [SYS]

6. **Oil depletes with use; ingenuity increases with use** — Bill's framing for the 10x paradigm shift. Every network built teaches the next one (Noelle signal loop). Every transaction sharpens Alice. The asset appreciates, not depreciates. This is the lead for capital conversations, before any numbers. Domain tag: [Cross-domain]

## Scars

- NHTSA API access: spent 20 minutes trying different User-Agent headers and endpoints before discovering Akamai blocks all programmatic access. Bulk CSV was the answer from the start. Cost: time. Lesson: when a government API returns 403, check for bulk download alternatives immediately.
- County FIPS mismatch: census auto-fetch silently returned 0 tracts for Asheville because the county code was wrong. No error, just empty data served from stale Greenville files. Cost: confusion, wasted fetch cycle. Lesson: validate FIPS codes by checking tract count > 0 before declaring success.
