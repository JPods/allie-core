## Scars This Session

**JSON brace vs bracket** — Bill drafted CCW using `{1, "track"}`. Invalid JSON — objects
require key:value pairs, not comma-separated values. Arrays `[1, "track"]` are correct.
Bill caught it himself: "I should have used [] instead of {}". Rule: when authoring JSON
manually, sets/tuples are always `[]`; `{}` is only for key:value maps.

**EP collapse when track removed** — When gw_lift_parking was confirmed absent from the
station_parking model, two EPs (lift_in→lift_parking and lift_parking→lift) had to collapse
into one. The renumbering cascades through all subsequent EPs. Rule: before collapsing eps,
enumerate ALL eps sequentially to catch cascade errors. Don't renumber in place — write
the full corrected array.

## Lessons for Allie

1. **Numbered CCW pairs are [pos, "track"], not {pos, "track"}** — JSON arrays only.
   When reviewing Bill's template drafts, check for brace/bracket confusion immediately.
   Domain: [SketchUp]

2. **CP-unit grouping is the CCW ordering rule** — out_lead → uturn → out_stub → in_stub → in_lead
   as a group, then interior tracks (platform, lift, main), then CP1 group. This is NOT a
   topological traversal — it is a human-readable grouping by function. Sally's BFS ordering
   does not match this. jpod_sally.rb needs updating.
   Domain: [SketchUp]

3. **Removing a track from lines{} requires EP collapse and full renumber** — When a track
   is absent from the physical model, remove it from lines{} AND remove both EPs that
   bracket it, collapsing them to a single EP with the combined in[]/out[]. Then renumber
   ALL subsequent EP ids sequentially. Check landing/exit chain switch records too — they
   reference EP ids that shift.
   Domain: [SketchUp]

4. **chains_header.approved_by is the Noelle gate** — Noelle refuses Build if chains_header
   missing or approved_by empty. Do not author chains and leave approved_by blank unless
   intentionally blocking Build. Set it when chains are complete and verified by Bill.
   Domain: [SketchUp]

5. **alpha ordering embeds CCW position number** — alpha array is alphabetical by track name,
   but each entry carries its CCW position: `[ccw_pos, "track_name"]`. This lets readers
   find any track's CCW position without scanning the CCW array.
   Domain: [SketchUp]
