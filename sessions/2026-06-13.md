## Scars This Session

**Component definition vs. instance entity** — Writing entity attributes to a
`find_vehicles`-found object (from `e.definition.entities`) modifies the shared component
definition, not the model instance. The change appears to work (no error, returns true)
but is invisible to animation code that reads from model-level instances. Cost: 2+ hours.
Rule: always use `all_nora_vehicles_in_model` or `model.entities` search to find instances.
Added to scars.md.

**Erase-recreate test harness invalidates all patches** — Any test that calls
model.entities.erase_entities + creates new entities on start discards all prior
attribute edits. There is no way to pre-patch entities that don't exist yet. The setup
lambda (tag_vehicle) is the ONLY place to set correct initial state.
Added to scars.md.
