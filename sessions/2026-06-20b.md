## Scars This Session

**Resume direction loss** — save_resume_state v1 saved only maneuver_id and t. On resume, raw lookup pts were used without direction info. Natalie reverses many intra-station tracks during trip build. Result: pods traveled backwards on reversed tracks. Cost: 2 debug cycles. Rule: any saved route bookmark must include direction info — the lookup is not the trip.

**Resume intercept order** — build_fleet processed Sally hold_loop BEFORE resume. After stop()/start(), Sally.reset cleared slot records. Every hold_loop pod hit "hl_slot unknown" → created parked pod at ps0 → `next` → resume never fired. The "falling through to normal dispatch" message was misleading — it fell into the catch-all, not through it. Rule: resume intercept must be FIRST — a pod with resume_state was traveling, period.

## Lessons for Allie

1. **Resume direction is not in the lookup** — Natalie reverses tracks during trip build. save_resume_state must save the start point of each maneuver as traveled. Resume compares saved start with lookup first/last pt to detect reversals. Without this, pods travel backwards on ~half the intra-station tracks. Domain: [SketchUp]

2. **Resume must fire before all other intercepts** — build_fleet order matters. hold_loop intercept consumes pods before resume can fire, especially after Sally.reset clears slot records. Rule: if resume_state exists on entity, that pod was traveling — process it first, skip everything else. Domain: [SketchUp]

3. **One guideway group, two map directions** — Build creates one physical guideway per connection. beam_path exists on that group. The reverse direction has no group, no beam_path. Any code that reads beam_path must check the reverse connection ID and use reversed pts. Domain: [SketchUp] [Cross-domain]

4. **Graceful stop needs proactive math** — When stopping, Natalie knows every destination. Don't wait for overflow — calculate demand (parked + inbound vs capacity) immediately and dispatch rebalancing trips before inbound pods arrive. Domain: [SketchUp] [Physical]
